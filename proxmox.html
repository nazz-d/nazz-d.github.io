What I Built

  • Built a Proxmox lab across enterprise rackmount and small form factor hardware.
  • Ran a mix of VMs and LXC containers for core services and experiments.
  • Hosted service categories including media, identity, dashboards, remote access, and game infrastructure.
  • Documented host inventory, service roles, ports, and likely dependencies.
  • Separated known facts from items still needing verification.

Operational Habits

  • Track host roles before making changes.
  • Use documentation to make troubleshooting repeatable.
  • Keep experiments separate from core services when possible.
  • Identify unknown hosts instead of ignoring them.
  • Use public-safe summaries for portfolio materials.
